Go to https://console.developers.google.com/
API & Auth --> Credentials
Two types of OAuth keys:
	* web application --> using google for authentication etc.
							NOTE: do not forget to register your callbacks under
							"Redirect URIs" and "Javascript Origins"

	* service account --> using google API to get files from google drive etc.

In sails, Authentication policies enforced under: config/policies.js
